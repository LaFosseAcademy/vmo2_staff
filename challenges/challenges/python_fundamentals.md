# Beginner Python Challenges (About 1 Hour)

These beginner-friendly Python exercises are designed to take around **60 minutes total**.  
Try to solve them in order. Each one builds on skills from the previous challenge.

## How to use this file
- Open Python in your editor or use GitHub Codespace
- Solve one challenge at a time
- Test your code with your own inputs
- Aim for working code before making it perfect

---

## Challenge 1: Print a Greeting

Write a program that prints:

```python
Hello, Python!
```

**Goal:** Learn how to display output with `print()`.

**Stretch:** Print your name on the next line.

---

## Challenge 2: Ask for a Name

Write a program that:
1. Asks the user for their name
2. Prints a message like:

```python
Hello, Sam!
```

**Goal:** Practice using `input()` and variables.

**Stretch:** Change the message to: `Welcome to Python, Sam!`

---

## Challenge 3: Add Two Numbers

Write a program that:
1. Asks the user for two numbers
2. Converts them into numbers
3. Prints the total

Example:

```python
Enter first number: 4
Enter second number: 7
Total: 11
```

**Goal:** Practice `input()`, `int()`, and arithmetic.

**Stretch:** Also print the product of the two numbers.

---

## Challenge 4: Even or Odd

Write a program that asks the user for a whole number and prints whether it is **even** or **odd**.

Example:

```python
Enter a number: 8
8 is even
```

**Goal:** Practice `if` statements and the modulo operator `%`.

**Hint:** A number is even if `number % 2 == 0`.

---

## Challenge 5: Simple Password Check

Create a program that asks the user to enter a password.

- If the password is `"python123"`, print `"Access granted"`
- Otherwise, print `"Access denied"`

**Goal:** Practice comparing strings with `if` / `else`.

**Stretch:** Let the user have 2 attempts.

---

## Challenge 6: Count from 1 to 10

Write a program that prints the numbers from 1 to 10 using a loop.

**Goal:** Practice using a `for` loop with `range()`.

**Stretch:** Print only the even numbers from 1 to 10.

---

## Challenge 7: Sum of 1 to N

Ask the user for a number `n`, then calculate the sum of all numbers from `1` to `n`.

Example:

```python
Enter a number: 5
Sum: 15
```

Because:

```python
1 + 2 + 3 + 4 + 5 = 15
```

**Goal:** Practice loops and running totals.

**Stretch:** Solve it without a loop.

---

## Challenge 8: List of Favorite Foods

Create a list with 3 of your favorite foods and then:

1. Print the whole list
2. Print the first item
3. Print the last item

Example:

```python
['pizza', 'tacos', 'pasta']
First: pizza
Last: pasta
```

**Goal:** Practice creating and using lists.

**Stretch:** Ask the user to add a fourth food with `input()`.

---

## Challenge 9: Count Vowels in a Word

Ask the user for a word and count how many vowels it contains.

Vowels are:

```python
a, e, i, o, u
```

Example:

```python
Enter a word: banana
Vowels: 3
```

**Goal:** Practice loops, strings, and counting.

**Stretch:** Make it work for uppercase letters too.

---

## Challenge 10: Mini Number Guessing Game

Create a program where:
- The secret number is `7`
- The user guesses a number
- The program prints:
  - `"Too low"` if the guess is smaller
  - `"Too high"` if the guess is larger
  - `"Correct!"` if the guess matches

**Goal:** Combine input, `if` statements, and comparison.

**Stretch:** Let the user keep guessing until they get it right.
